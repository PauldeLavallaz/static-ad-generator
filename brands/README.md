# Brands Folder

This folder holds all brand-specific assets and generated outputs. Each brand gets its own subfolder.

## Folder Structure

Create a subfolder for each brand you want to generate ads for:

```
brands/
└── your-brand-name/
    ├── product-images/          # Drop product PNGs/JPGs here before running
    │   ├── product-front.png
    │   ├── product-angle.png
    │   └── ...
    ├── brand-dna.md             # Generated by Phase 1 (Brand Research)
    ├── prompts.json             # Generated by Phase 2 (Prompt Generation)
    ├── generate_ads.py          # Copied from skills/references/ for Phase 3
    └── outputs/                 # Generated images organized by template
        ├── 01-headline/
        │   ├── headline_v1_img1.png
        │   └── prompt.txt
        ├── 02-offer-promotion/
        │   └── ...
        └── ...
```

## Getting Started

1. Create a folder with your brand name: `mkdir -p brands/your-brand/product-images/`
2. Drop 1-3 product images (PNG/JPG) into `product-images/`
3. Run the skill from the project root — Claude Code handles the rest
